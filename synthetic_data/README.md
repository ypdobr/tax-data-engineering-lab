# Synthetic Data

This folder will contain generated data only.

Rules:

- fictional companies
- fictional countries and entity structures
- fictional product and transaction IDs
- artificial amounts
- no real client values
- no copied schemas from work

Raw synthetic datasets should stay small enough for public review.

Schema documentation:

- `schemas/erp_source_schema.md`

Generated CSV output belongs in `data/` and is ignored by Git.
